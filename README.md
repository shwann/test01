# test01 公司门户首页

这是一个无外部依赖的公司门户首页示例，包含导航、首屏、公司介绍、产品服务、企业动态、数据概览和联系页脚。

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

也可以使用内置静态服务器：

```bash
npm start
```

默认访问地址：

```text
http://localhost:4173
```

如需指定端口：

```bash
PORT=3000 npm start
```

## 检查

```bash
npm run check
```

该命令会检查 JavaScript 文件语法，并验证首页关键区块与响应式样式是否存在。
